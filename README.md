### 概要

Sudoku One（以下「本アプリ」）は個人開発者によって開発・運営されています。このプライバシーポリシーでは、本アプリをご利用の際に収集されるデータ、および収集されないデータについて説明します。

---

### 収集するデータ

本アプリは以下のサードパーティサービスを使用しており、これらのサービスが自動的にデータを収集する場合があります。

**Google AdMob（広告配信）**
- 広告識別子（iOS の IDFA / Android の GAID）
- デバイスの種類および OS バージョン
- 広告インタラクションデータ（インプレッション、クリック）
- これらのデータは関連性の高い広告を表示する目的のみに使用されます。
- 詳細は [Google のプライバシーポリシー](https://policies.google.com/privacy) をご参照ください。

**クラッシュログ / 診断情報**
- バグの特定・修正のため、プラットフォーム（Apple / Google）が匿名のクラッシュレポートを収集する場合があります。
- これらのレポートには個人を特定できる情報は含まれません。

---

### 収集しないデータ

本アプリは以下のデータを一切収集・保存しません：

- 氏名、メールアドレスその他の個人識別情報
- 位置情報（GPS・ネットワークベースを含む）
- 写真、連絡先、その他のデバイスコンテンツ
- 金融・決済情報（アプリ内購入は Apple / Google が完全に処理します）
- 個人に紐付いた利用履歴

ゲームの進行状況および設定は**お使いのデバイス内のみ**に保存され、外部サーバーに送信されることはありません。
